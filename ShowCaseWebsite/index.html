Features

Search Form

Here the user can enter a specific topic to be analysed and specific parameters for it. The user can search by the following cateogirs:

  • Topic - Enter a specifc topic and posts that mention it will be analysed
  • User - Analysis of a specifc redditor
  • Subreddit - Analysis of any subreddit on the platform
  • Comments - Analysis of a comment tree on a specifc post
  • Domain - Analyis of posts that link to a specific domain
This analysis can be used to gauge the public sentiment towards a certain topic. It can also be used to verify how toxic an account or a community is.

Google Trending Topics

Here the user can check what is currently trending on google in those 4 regions, and then go and check the sentiment of the topic on Reddit.

Charts

Here the user can view the sentiment of the topic in a visual format. The user can view the sentiment of the topic over time, the breakdown on a pie chart the engagement vs sentiment, etc.

View Posts

In this section the user can view each individual post that was analysed. Its sentiment, the number of comments, the number of upvotes, as well as much more.
